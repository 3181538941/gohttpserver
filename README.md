# gohttp-vue
Rewrite https://github.com/codeskyblue/gohttp with golang+vue

## Not ready yet.

## Notes
If using go1.5, ensure you set GO15VENDOREXPERIMENT=1

## Installation
```
go get -v github.com/kardianos/govendor
go get -v github.com/codeskyblue/gohttp-vue
```

## Usage
```
./gohttp-vue -p 8000
```
